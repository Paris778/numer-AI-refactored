## Introduction

The Numerai Data Science Tournament is a unique machine learning competition focused on predicting stock market performance using obfuscated financial data. This guide consolidates all essential knowledge for building, evaluating, and deploying models.

### Key Concepts

- **Obfuscated Data**: Stock IDs, feature names, and target definitions are anonymized
- **No Domain Knowledge Required**: Models can be built without financial expertise
- **Weekly Submissions**: New predictions required Tuesday-Saturday
- **Staking**: Stake NMR cryptocurrency on model performance

## Data Structure

### Eras

- **Era**: Represents a specific date (weekly intervals)
- Each era contains multiple stocks
- Treat all stocks within an era as a single training example
- **Important**: Always calculate metrics per-era, never flatten all predictions

```python
# Load training data
train = pd.read_parquet(
    f"{DATA_VERSION}/train.parquet",
    columns=["era", "target"] + feature_set
)

# Check era distribution
train.groupby("era").size().plot(
    title="Number of rows per era",
    xlabel="Era"
)
```

### Data Splits

| Split | Description | Usage |
|-------|-------------|-------|
| **Train** | Historical eras | Model training |
| **Validation** | Recent out-of-sample eras | Performance evaluation |
| **Test** | Future eras (not yet available) | Tournament scoring |
| **Live** | Current week | Generate weekly submissions |

### Embargo Period

**Critical**: When using validation data, embargo the first 4 eras after your last training era to prevent data leakage.

```python
# Embargo overlapping eras
last_train_era = int(train["era"].unique()[-1])
eras_to_embargo = [str(era).zfill(4) for era in [last_train_era + i for i in range(4)]]
validation = validation[~validation["era"].isin(eras_to_embargo)]
```

**Why 4 eras?** Targets look 20 days (~4 weeks) into the future, so validation eras within this window would contain information from training data.

**Key Statistics**:
```python
def get_summary_metrics(scores, cumsum_scores):
    mean = scores.mean()
    std = scores.std(ddof=0)
    sharpe = mean / std
    rolling_max = cumsum_scores.expanding(min_periods=1).max()
    max_drawdown = (rolling_max - cumsum_scores).max()
    
    return {
        "mean": mean,
        "std": std,
        "sharpe": sharpe,
        "max_drawdown": max_drawdown
    }

summary = get_summary_metrics(per_era_corr, cumsum_corr)
```

**What to Optimize**:
- **Mean**: Overall performance
- **Sharpe**: Consistency (mean/std)
- **Max Drawdown**: Risk management

## Feature Neutralization

### What is Feature Neutralization?

**Purpose**: Reduce unwanted feature exposure to improve model stability

**How it Works**: Orthogonalize predictions with respect to features using least-squares regression

### Why Neutralize?

1. **Reduce Feature Risk**: Features may lose predictive power or reverse over time
2. **Minimize Unintended Exposures**: Avoid overexposure to market-wide risk factors
3. **Improve Stability**: More consistent performance across different market regimes

### Measuring Feature Exposure

```python
# Calculate per-era correlation between predictions and features
feature_exposures = validation.groupby("era").apply(
    lambda d: d[features].corrwith(d["prediction"])
)

# Plot exposures
feature_exposures.plot.bar(
    title="Feature Exposures",
    figsize=(16, 10),
    subplots=True,
    layout=(7, 5),
    xticks=[],
    legend=False
)

# Mean maximum feature exposure
max_feature_exposure = feature_exposures.abs().max(axis=1)
print("Mean max feature exposure:", max_feature_exposure.mean())
```

### Neutralization Implementation

```python
def neutralize(
    df: pd.DataFrame,
    neutralizers: pd.DataFrame,
    proportion: float = 1.0,
) -> pd.DataFrame:
    """
    Neutralize predictions to features using least-squares regression.
    
    Args:
        df: DataFrame with predictions to neutralize
        neutralizers: DataFrame with features to neutralize against
        proportion: Degree of neutralization (0.0 to 1.0)
        
    Returns:
        DataFrame with neutralized predictions
    """
    assert not neutralizers.isna().any().any(), "Neutralizers contain NaNs"
    assert len(df.index) == len(neutralizers.index), "Indices don't match"
    
    # Handle zero-variance columns
    df[df.columns[df.std() == 0]] = np.nan
    
    df_arr = df.values
    neutralizer_arr = neutralizers.values
    
    # Add intercept column
    neutralizer_arr = np.hstack((
        neutralizer_arr, 
        np.array([1] * len(neutralizer_arr)).reshape(-1, 1)
    ))
    
    # Compute orthogonal projection
    inverse_neutralizers = np.linalg.pinv(neutralizer_arr, rcond=1e-6)
    adjustments = proportion * neutralizer_arr.dot(inverse_neutralizers.dot(df_arr))
    neutral = df_arr - adjustments
    
    return pd.DataFrame(neutral, index=df.index, columns=df.columns)
```

### Applying Neutralization

#### Per-Era Neutralization (Recommended)

```python
from numerai_tools.scoring import neutralize

# Neutralize per era
neutralized = validation.groupby("era", group_keys=False).apply(
    lambda d: neutralize(
        d[["prediction"]],
        d[features],
        proportion=1.0
    )
).reset_index().set_index("id")

validation["prediction_neutral"] = neutralized["prediction"]
```

#### Partial Neutralization

```python
# Neutralize at different proportions
proportions = [0.25, 0.5, 0.75, 1.0]

for proportion in proportions:
    neutralized = validation.groupby("era", group_keys=False).apply(
        lambda d: neutralize(
            d[["prediction"]],
            d[features],
            proportion=proportion
        )
    ).reset_index().set_index("id")
    
    validation[f"neutral_{proportion*100:.0f}"] = neutralized["prediction"]
```

### Group Neutralization

Neutralize to feature groups instead of individual features:

```python
# Neutralize to specific feature group
group = "serenity"  # or "intelligence", "charisma", etc.
group_features = list(subgroups["small"][group])

neutralized = validation.groupby("era", group_keys=False).apply(
    lambda d: neutralize(
        d[["prediction"]],
        d[group_features],
        proportion=1.0
    )
).reset_index().set_index("id")

validation[f"neutral_{group}"] = neutralized["prediction"]
```

### Verifying Neutralization

```python

# Should be approximately zero
```

### Trade-offs

**Pros**:
- Reduced feature risk
- Better max drawdown (more stable)
- Lower unintended exposures

**Cons**:
- May reduce mean correlation
- Removes some signal along with risk
- Requires careful feature selection

**Decision Framework**:
```python
# Compare before/after
if neutral_sharpe > original_sharpe:
    print("Neutralization improves risk-adjusted returns")
elif neutral_max_drawdown < original_max_drawdown:
    print("Neutralization reduces risk but at cost of returns")
else:
    print("Neutralization not beneficial for this model")
```

## Target Ensembling

### Why Ensemble Targets?

1. **Diversification**: Different targets capture different market aspects
2. **Robustness**: Reduces overfitting to single target definition
3. **Stability**: Smoother performance across market regimes

### Target Analysis

```python
# Examine target correlations
target_corr = targets_df[target_cols].corr()

# Correlation with main target
main_target_corr = targets_df[target_cols].corrwith(targets_df[MAIN_TARGET])
main_target_corr.sort_values(ascending=False)
```

**Key Insight**: 
- High correlation (0.8+): Similar predictions
- Low correlation (<0.5): Diverse perspectives → better for ensembling

### Training Multiple Models

```python
# Train model per target

TARGET_CANDIDATES = [
    "target_ender_20",
    "target_victor_20",
    "target_xerxes_20",

for target in TARGET_CANDIDATES:
```

### Generating Ensemble Predictions

### Ensemble Methods

#### 1. Simple Average (Rank-based)

```python
# Rank predictions within each era, then average
validation["ensemble_avg"] = (
    validation
    .groupby("era")[pred_cols]
    .rank(pct=True)
    .mean(axis=1)
)
```

#### 2. Weighted Average

```python
# Weight by validation performance
weights = {
    "prediction_target_ender_20": 0.4,
    "prediction_target_victor_20": 0.3,
    "prediction_target_xerxes_20": 0.2,
    "prediction_target_teager2b_20": 0.1
}

validation["ensemble_weighted"] = 0
for pred_col, weight in weights.items():
    validation["ensemble_weighted"] += (
        validation.groupby("era")[pred_col].rank(pct=True) * weight
    )
```

#### 3. Pairwise Ensembles

```python
# Ensemble specific target pairs
validation["ensemble_ender_victor"] = (
    validation
    .groupby("era")[[
        "prediction_target_ender_20",
        "prediction_target_victor_20"
    ]]
    .rank(pct=True)
    .mean(axis=1)
)

validation["ensemble_ender_teager"] = (
    validation
    .groupby("era")[[
        "prediction_target_ender_20",
        "prediction_target_teager2b_20"
    ]]
    .rank(pct=True)
    .mean(axis=1)
)
```

```python
    live_features: pd.DataFrame,
    live_benchmark_models: pd.DataFrame = None
) -> pd.DataFrame:
    # Generate predictions from each model
    predictions = pd.DataFrame(index=live_features.index)
    
    for target in TARGET_CANDIDATES:
        predictions[target] = models[target].predict(live_features[features])
    
    # Ensemble via ranking
    ensemble = predictions.rank(pct=True).mean(axis=1)
    
    # Format submission
    submission = ensemble.rank(pct=True, method="first")
    return submission.to_frame("prediction")
```

```python
def check_stability(correlations, min_sharpe=0.5, max_drawdown=0.2):
    """Check if model meets stability criteria"""
    mean = correlations.mean()
    std = correlations.std(ddof=0)
    sharpe = mean / std
    
    cumsum = correlations.cumsum()
    drawdown = (cumsum.expanding(min_periods=1).max() - cumsum).max()
    
    checks = {
        "Sharpe >= {min_sharpe}": sharpe >= min_sharpe,
        "Max Drawdown <= {max_drawdown}": drawdown <= max_drawdown,
        "Positive mean": mean > 0
    }
    
    for check, passed in checks.items():
        status = "✓" if passed else "✗"
        print(f"{status} {check}")
    
    return all(checks.values())
```

```python
def get_summary_metrics(correlations):
    """Calculate comprehensive performance metrics"""
    mean = correlations.mean()
    std = correlations.std(ddof=0)
    sharpe = mean / std
    cumsum = correlations.cumsum()
    max_drawdown = (cumsum.expanding(min_periods=1).max() - cumsum).max()
    
    # Half-split comparison
    n = len(correlations)
    first_half = correlations.iloc[:n//2].mean()
    second_half = correlations.iloc[n//2:].mean()
    delta = abs(first_half - second_half)
    
    return {
        "mean": mean,
        "std": std,
        "sharpe": sharpe,
        "max_drawdown": max_drawdown,
        "delta": delta
    }


def apply_embargo(train_df, validation_df, embargo_eras=4):
    """Apply embargo period between train and validation"""
    last_train_era = int(train_df["era"].unique()[-1])
    eras_to_embargo = [
        str(era).zfill(4) 
        for era in range(last_train_era + 1, last_train_era + embargo_eras + 1)
    ]
    return validation_df[~validation_df["era"].isin(eras_to_embargo)]


def calculate_feature_exposures(df, features, prediction_col="prediction"):
    """Calculate per-era feature exposures"""
    return df.groupby("era").apply(
        lambda d: d[features].corrwith(d[prediction_col])
    )


def create_ensemble(df, prediction_cols, method="rank_mean"):
    """Create ensemble from multiple predictions"""
    if method == "rank_mean":
        return (
            df
            .groupby("era")[prediction_cols]
            .rank(pct=True)
            .mean(axis=1)
        )
    elif method == "simple_mean":
        return df[prediction_cols].mean(axis=1)
    else:
        raise ValueError(f"Unknown method: {method}")


def validate_submission(predictions, live_features):
    """Validate submission format"""
    checks = {
        "'prediction' column exists": "prediction" in predictions.columns,
        "Index matches": predictions.index.equals(live_features.index),
        "No NaN values": not predictions.isna().any().any(),
        "Row count matches": len(predictions) == len(live_features)
    }
    
    all_passed = True
    for check, passed in checks.items():
        status = "✓" if passed else "✗"
        print(f"{status} {check}")
        if not passed:
            all_passed = False
    
    return all_passed
```
